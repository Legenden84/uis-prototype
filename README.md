## Vejledning:
Såfremt at Python3.8 er installeret. Kan følgende trin udføres.

1) Installer dependencies: \
    $ pip install dash==2.8.1
    $ pip install dash_daq==0.5.0

2) Kør Aula-prototypen: \
    $ python app.py
